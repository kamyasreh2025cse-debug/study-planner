# 📚 Study Planner

## 📌 Description
A Python-based application that helps students generate an optimized study schedule based on subject priority, available time, and workload.

## ✨ Features
- Accepts subject names, study hours, and priority levels  
- Sorts subjects based on study hours and importance  
- Displays an optimized study plan  
- Calculates total study time  
- Suggests breaks for better productivity  

## 🛠 Technologies Used
- Python  

## ▶️ How to Run
1. Clone the repository:
   git clone https://github.com/kamyasreh2025cse-debug/study-planner.git
2. Navigate to the folder:
   cd study-planner
3. Run the program:
   python study_planner.py
